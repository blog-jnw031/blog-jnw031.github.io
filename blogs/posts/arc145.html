ARC 145

This morning, I participated in AtCoder Regular Contest 145 and here's my solutions and feelings about the contest.

A

Problem Link.

It takes me a lot of time to find the properties of the problem, and I finally found that if the last letter is \(\texttt{A}\), I can change all the previous letters to \(\texttt{ABB...B}\) to make it a palindrome. If both the first and the last letter is \(\texttt{B}\), I can change the string \(\texttt{B****B}\) to \(\texttt{BAA...AB}\) and it's also a palindrome.

Therefore, the only impossible case is when \(S_1=\texttt{A}\) and \(S_n=\texttt{B}\), but I missed a special case so I got a penalty: when \(n=2\), the answer is yes if and only if \(S_1=S_2\).

Problem Link.

I thought the answer when \(a\geq n\) and \(a\ < b\) is \[b\left(\left\lfloor\frac{n+1}{a}\right\rfloor+1\right)+\min\{\left(n+1\right)\bmod a,b\}\] and I thought it was fake, after the end of the contest, I sent a post for help and found that I accidently wrote an \(\texttt{if}\) instead of \(\texttt{else if}\), and my formula is correct.
